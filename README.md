# Robotics AI Project 🚀

## Overview

This project focuses on building an autonomous robot capable of navigating complex environments and locating humans or animals using multiple sensors and artificial intelligence.

The goal is to combine robotics, embedded systems, and machine learning into a real-world prototype.

---

## Objectives

* Develop a robot that can explore and map unknown environments
* Detect humans and animals using camera and thermal imaging
* Navigate through cluttered environments (e.g. basement prototype)
* Handle difficult conditions such as low visibility, fog, and obstacles

---

## Technologies

* ROS2 (Robot Operating System)
* Python & C++
* Deep Learning (computer vision)
* Reinforcement Learning (decision-making)
* Sensor Fusion (camera, thermal, distance sensors)

---

## System Architecture

1. Sensors collect environmental data
2. Perception system processes data using AI
3. Mapping system builds a representation of the environment (SLAM)
4. Planning system determines optimal path
5. Control system executes movement

---

## Prototype Plan

The first prototype will be tested in a cluttered indoor environment (e.g. basement) to simulate real-world complexity.

---

## Learning Goals

* Understand ROS2 architecture
* Learn Deep Learning and Reinforcement Learning
* Build a full robotics system from scratch
* Integrate hardware and software
* Gain practical engineering experience

---

## Vision

Long-term goal: Develop an autonomous search-and-rescue system for real-world disaster scenarios.
